# Sistema de Batalha em Turnos

Projeto simples de sistema de batalha em turnos desenvolvido em C# para praticar Programação Orientada a Objetos (POO), lógica de jogos e interação via terminal.

## Funcionalidades

- Sistema de batalha em turnos
- Ataque com dano aleatório
- Sistema de cura
- Verificação de HP máximo
- Inimigo controlado por IA simples
- Sistema de derrota
- Interface organizada no terminal

## Tecnologias Utilizadas

- C#
- .NET
- Console Application

## Conceitos Praticados

- Classes e Objetos
- Encapsulamento
- Métodos
- Propriedades
- Construtores
- Random
- Estruturas condicionais
- Loops
- Lógica de gameplay

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/therealdoubler312/sistema-de-batalha-em-turnos.git
```

2. Abra o projeto no Visual Studio

3. Execute o projeto

## Exemplo de Gameplay

```txt
Jogador HP = 100 Inimigo HP = 75
Turno do Jogador! O que você irá fazer? [A] Atacar [C] Curar
```
